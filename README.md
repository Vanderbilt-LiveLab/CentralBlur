# CentralBlur
### Setup Instructions

1. **Install Git LFS** (only once per machine)

   - **macOS:**
     ```bash
     brew install git-lfs
     ```
   - **Windows:**
     ```bash
     git lfs install
     ```

    
2. **Clone this repo.
3. **Open Scene: Snellen Scene
4. Build and run on Varjo Aero
